# 🕌 Halal Restaurant Finder Finland

A React web app that displays halal restaurants across Finland on an interactive map.

🌍 **Live Demo:** https://halal-restaurant-finder-production.up.railway.app/

## Features
- Interactive map of Finland with restaurant pins
- Click any pin to see restaurant details
- Search by restaurant name or city
- Filter by city and cuisine type
- Data fetched from Google Sheets CSV

## Tech Stack
- React + Vite
- React Leaflet (map)
- PapaParse (CSV parsing)
- Deployed on Railway

## Run Locally

1. Clone the repo:
   git clone https://github.com/Ahmed-Imran-khan/halal-restaurant-finder.git

2. Install dependencies:
   npm install

3. Start the dev server:
   npm run dev

4. Open http://localhost:5173
